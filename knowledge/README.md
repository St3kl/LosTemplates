# LosTemplates Knowledge Base

## Purpose

The Knowledge Base is a structured learning library that accompanies the LosTemplates project.

Its goal is to explain the technologies, concepts, and engineering principles used throughout the application.

Unlike the `docs/` directory, which documents the LosTemplates project itself, the Knowledge Base teaches the underlying theory and practical skills needed to understand and build it.

## Objectives

- Learn software engineering from first principles.
- Understand why technologies exist before learning how to use them.
- Connect theory with real implementation in LosTemplates.
- Build a long-term reference library for future projects.
